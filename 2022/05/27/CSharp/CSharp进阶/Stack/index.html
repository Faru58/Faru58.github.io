Stack

Stack是C#已经封装好的一个类
它的本质是一个object类型的数组,只是封装了特殊的存储规则

Stack是一个栈存储容器，栈是一种先进后出的数据结构
先存入的数据（栈底）后获取，后存入的数据（栈顶）先获取

声明

using System.Collections;
    Stack stack = new Stack();

增(压栈)

//往栈顶压入一个元素
stack.Push(object);

取(弹栈)

栈里不存在删除的概念

//弹出栈顶元素
object = stack.Pop();

查

栈无法查看指定位置的元素，只能查看栈顶的内容

//查看栈顶的内容(并不弹出来)
object = stack.Peek();
//查看元素是否存在与栈中
stack.Contains(object);

改

栈无法改变其中元素，它只能压和弹，但可以清空

//清空栈
stack.Clear();

遍历

由于栈里面没有提供索引器来中括号访问的接口，所以无法用for直接进行循环遍历

//栈长度
stack.Count

//迭代器遍历 顺序栈顶到栈底
foreach (object item in stack)

装箱拆箱

stack本质上是一个有特殊存储规则的object数组
由于用万物之父object来存储数据，自然存在装箱拆箱。
当往其中进行值类型存储时即为装箱，当将值类型对象取出来时即为拆箱

int i = 1;
stack.Push(i); //装箱
i = (int)stack.Pop(); //拆箱

using System;
using System.Collections;

namespace Lesson2_Stack
{
    internal class Program
    {
        static void Main(string[] args)
        {
            while (true)
            {
                Console.Write("请输入一个十进制数: ");
                uint num = uint.Parse(Console.ReadLine());
                Console.Write("转换成的二进制数: ");
                Calc(num);
                Console.WriteLine("\n");
            }
            
        }

        static void Calc(uint num)
        {
            Stack stack = new Stack();
            while(num > 1)
            {
                stack.Push(num % 2);
                num /= 2;
                if(num == 1)
                {
                    stack.Push(num);
                    break;
                }
            }
            while(stack.Count > 0)
            {
                Console.Write(stack.Pop());
            }
        }
    }
}
