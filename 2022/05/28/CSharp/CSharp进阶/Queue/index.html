Queue

Queue是C#已经封装好的一个类
它的本质是一个object类型的数组,只是封装了特殊的存储规则

Queue是队列存储容器，
队列是一种先进先出的数据结构，
先存入的数据先获取，后存入的数据后获取

声明

using System.Collections;
    Queue queue = new Queue();

增

//往队列中存储一个元素
queue.Enqueue(object);

取

队列里不存在删除的概念

//取出先加入队列的元素
object = queue.Dequeue();

查

队列无法查看指定位置的元素，只能查看最后加入队列的内容

//查看最后加入队列的元素
object = queue.Peek();
//查看元素是否存在队列中(bool)
stack.Contains(object);

改

队列无法改变其中元素，它只能进出队列，但可以清空

//清空栈
stack.Clear();

遍历

由于队列里面没有提供索引器来中括号访问的接口，所以无法用for直接进行循环遍历

//栈长度
queue.Count

//迭代器遍历 顺序栈顶到栈底
foreach (object item in queue)
{
    //需要的逻辑 
}

//将栈转换为object数组 顺序栈顶到栈底
object[] array = queue.ToArray();
for( i = 0; i < array.Length; i++ )
{
    //需要的逻辑 
}

//循环出列
while( queue.Count > 0 )
{
    object q = queue.Dequeue();
    Console.WriteLine(q);
}
Console.WriteLine( queue.Count );

装箱拆箱

queue本质上是一个有特殊存储规则的object数组
由于用万物之父object来存储数据，自然存在装箱拆箱。
当往其中进行值类型存储时即为装箱，当将值类型对象取出来时即为拆箱

int i = 1;
queue.Enqueue(i); //装箱
i = (int)queue.Dequeue(); //拆箱

题2

using System;
using System.Collections;

namespace Lesson3_Queue
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Queue queue = new Queue();
            Console.WriteLine("请输入你要存储的消息：");
            Input(queue);
            Console.WriteLine("\n已保存您的消息，按回车为您播放\n");
            if( Console.ReadKey(true).Key == ConsoleKey.Enter )
            {
                Output(queue);
                Console.WriteLine("\n已播放完所有您的消息");
                Environment.Exit(0);
            }
        }

        static void Input(Queue queue)
        {
            object message;
            //输入10个数停止输入（可以拓展为特殊指令停止输入）
            while ((string)(message = Console.ReadLine()) != "end")
            {
                queue.Enqueue(message);
            }
        }

        static void Output(Queue queue)
        {
            int updateIndex = 0;
            while (queue.Count > 0)
            {
                if (updateIndex++ == 500000000)
                {
                    object q = queue.Dequeue();
                    Console.WriteLine(q);
                    updateIndex = 0;
                }
            }
        }
    }
}
