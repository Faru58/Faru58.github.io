泛型

泛型

概念

泛型实现了类型参数化，打到代码重用的目的
通过类型参数化来实现同一份代码上操作多种类型

泛型相当于类型占位符
定义类或方法时使用替代符代表变量类型
当真正使用类或者方法时再具体指定类型

泛型分类

泛型类和泛型接口
基本语法：

class 类名<泛型占位字符>
interface 接口名<泛型占位字符>

泛型函数
基本语法：

函数名<泛型占位字符>(参数列表)

注意：
泛型占位字母可以有多个，用逗号分开

泛型分类实例

class TestClass<T>
{
    public T value;
}
class TestClass<T1,T2,K,M,Key>
{
    //每个占位符
}

TestClass<int> t = new TestClass<int>();
interface TestInterface<T>
{
    T value
    {
        get;
        set;
    }
}

class Test : TestInterface<int>
{
    public int value
    {
        get;
        set;
    }
}

泛型方法实例

普通类中的泛型方法

class Test2
{
    public void TestFun<T>( T value )
    {
        Console.WriteLine(value);
    }
}

Test2 test2 = new Test2();
test2.TestFun<string>("123123");

泛型类中的泛型方法

class Test2<T>
{
    public T value;

    //这不是一个泛型方法，因为T是泛型类声明的时候就指定，在使用这个函数的时候不能动态的改变T
    public void TestFun(T t)
    {

    }

    //这是一个泛型方法，T已经被泛型类使用，泛型方法中不能用T
    public void TestFun<K>( K value )
    {
        Console.WriteLine(value);
    }
}

Test2 test2 = new Test2();
test2.TestFun<string>("123123");

泛型的作用

1.不同类型对象的相同逻辑处理就可以选择泛型
2.使用泛型可以一定程度避免装箱拆箱

练习题

using System;

namespace Lesson5_泛型
{
    class Type
    {
        public void WhatType<T> ()
        {
            if (typeof(T) == typeof(int))
            {
                Console.WriteLine("整形, 4字节");
            }
            else if(typeof(T) == typeof(char))
            {
                Console.WriteLine("字符, 1字节");
            }
            else if (typeof(T) == typeof(float))
            {
                Console.WriteLine("单精度浮点数, 8字节");
            }
            else if (typeof(T) == typeof(string))
            {
                Console.WriteLine("字符串, 不定字节数");
            }
            else
            {
                Console.WriteLine("其他类型");
            }

        }
    }
    internal class Program
    {
        static void Main(string[] args)
        {
            Type type = new Type();
            type.WhatType<int>();
        }
    }
}
