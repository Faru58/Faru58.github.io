类和对象

类 对象

类

基本概念：
1.具有相同特征
2.具有相同行为
3.一类事物的抽象
4.类是对象的模板
5.可以通过类创建出对象
6.类的关键词 class

一般声明在 namespace 语句块中
声明语法：

访问修饰符 class 类名
{
    //封装
    特征 成员变量
    行为 成员方法
    保护特征 成员属性

    构造函数和析构函数
    索引器
    运算符重载
    静态成员
}

（类）对象

基本概念：
1.类的声明和类对象（变量）声明是两个概念
2.类的声明类似枚举和结构体的声明（类的声明相当于声明了一个自定义变量类型）
3.对象是类创建出来的（相当于声明一个指定类的变量）
4.类创建对象的过程（一般称为实例化对象）
5.类对象都是引用类型的

声明语法：

类名 变量名;
类名 变量名 = null; //null代表空
类名 变量名 = new 类名();

总结

1.类的声明和类对象（变量）声明是两个概念
2.类的声明是声明对象的模板，用来抽象（形容）显示事物的
3.类对象（变量）声明是用来表示现实中的对象个体的

4.类是一个自定义的变量类型
5.实例化一个类对象是在声明变量
