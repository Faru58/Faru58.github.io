字符串指定位置获取

字符串本质是char数组

string str = "唐老狮";
Console.WriteLine(str[0]); //打印“唐”

可以转换化为char数组

char[] chars = str.ToCharArray();
Console.WriteLine(chars[1]); //打印“老”

字符串替换

Format(); 后接变长参数

string str = string.Format("{0}{1}", 1, 3333);
Console.WriteLine(str); //打印13333

正向查找字符位置

IndexOf(); 返回字符位置

string str = "我是唐老狮！";
int index = str.IndexOf("唐");
Console.WriteLine(index); //打印1

若没找到则返回-1
