抽象类和抽象函数

抽象类和抽象函数

抽象类和抽象函数

抽象类

概念：
被抽象关键字abstract修饰的类
不希望被实例化的对象,相对比较抽象的类可以使用抽象类（水果等）

特点：
1.不能被实例化的类，但遵循里式替换原则（用父类容器装子类）
2.可以包含抽象方法
3.继承抽象类必须重写其抽象方法

抽象函数

又称 纯虚方法
用abstract关键字修饰的方法

特点：
1.只能在抽象类中声明
2.没有方法体
3.不能是私有的
4.继承后必须实现 用override重写

虚方法和抽象方法的异同：
相同点：
可以无限被子类重写
不同点：
1.抽象方法需要调用,必须用public或protected修饰,不能有函数体,只能在子类里override重写；虚方法一定有函数体,且里面可写可不写逻辑
2.抽象方法继承后必须实现；虚方法是子类里选择性实现的

abstract class Fruits
{
    public string name;

    //因为抽象函数需要调用,必须用public或protected修饰
    //不能有函数体,只能在子类里override重写
    abstract public void Bad();    

    public virtual void Test()
    {
        //虚方法一定有函数体,且里面可写可不写逻辑
    }
}

class Apple : Fruits
{
    //虚方法是子类里选择性实现的
    //抽象方法继承后必须实现
    public override void Bad()
    {
        Console.WriteLine("苹果坏了");
    }
}

练习题

题1

using System;

namespace Lesson18_抽象类和抽象函数
{
    abstract class Animals
    {
        abstract public void Speak();    

    }

    class People : Animals
    {
        public override void Speak()
        {
            Console.WriteLine("人叫");
        }
    }

    class Dog : Animals
    {
        public override void Speak()
        {
            Console.WriteLine("狗叫");
        }
    }

    class Cat : Animals
    {
        public override void Speak()
        {
            Console.WriteLine("猫叫");
        }
    }

    internal class Program
    {
        static void Main(string[] args)
        {
            Animals p = new People();
            p.Speak();
            Animals d = new Dog();
            d.Speak();
            Animals c = new Cat();
            c.Speak();
        }
    }
}

题2

using System;

namespace Lesson18_抽象类和抽象函数
{
    //图形
    abstract class Figure
    {
        //求面积
        public abstract void Area();
        //求周长
        public abstract void Perimeter();
    }

    //矩形
    class Rectangular : Figure
    {
        public int l;
        public int w;
        public Rectangular(int l, int w)
        {
            this.l = l;
            this.w = w;
        }
        public override void Area()
        {
            int s = l * w;
            Console.WriteLine("该矩形面积为{0}", s);
        }
        public override void Perimeter()
        {
            int c = 2 * (l + w);
            Console.WriteLine("该矩形周长为{0}", c);
        }
    }

    //矩形
    class Square : Figure
    {
        public int a;
        public Square(int a)
        {
            this.a = a;
        }
        public override void Area()
        {
            int s = a * a;
            Console.WriteLine("该矩形面积为{0}", s);
        }
        public override void Perimeter()
        {
            int c = 4 * a;
            Console.WriteLine("该矩形周长为{0}", c);
        }
    }

    //圆形
    class Circular : Figure
    {
        public int r;
        const float PI = 3.1415926f;
        public Circular(int r)
        {
            this.r = r;
        }
        public override void Area()
        {
            float s = PI * r * r;
            Console.WriteLine("该矩形面积为{0}", s);
        }
        public override void Perimeter()
        {
            float c = 2 * r * PI;
            Console.WriteLine("该矩形周长为{0}", c);
        }
    }


    internal class Program
    {
        static void Main(string[] args)
        {
            Figure r = new Rectangular(2, 4);
            r.Area();
            r.Perimeter();
            Figure s = new Square(3);
            s.Area();
            s.Perimeter();
            Figure c = new Circular(5);
            c.Area();
            c.Perimeter();
        }
    }
}
