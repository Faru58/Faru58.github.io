输入相关内容写在Update内

鼠标在屏幕位置

Input.mousePosition

屏幕坐标的原点在屏幕的左下角，往右是x轴正向形，往上是y轴正方向
返回值是Vector3，但是只有x和y有值，z一直是0，这是因为屏幕本来就是2D的，不存在z轴

检测鼠标输入

鼠标按下一瞬间进入

//只有按下的这一瞬间进入

{
    //逻辑
}

鼠标抬起一瞬间进入

//0左键 1右键 2中键
//只有抬起的这一瞬间进入

    //逻辑
}

//0左键 1右键 2中键
//当按住不放时，会一直进入这个判断

    //逻辑
}

//返回值是Vector3,返回值的y=-1表示往下滚，y=0表示没有滚，y=1表示向上滚

检测默认轴输入

键盘AD按下时，返回-1到1之间的变换

//相当于得到的这个值，就是我们的左右方向，通过他来控制对象左右移动或旋转
Input.GetAxis("Horizontal");

键盘SW按下时，返回-1到1之间的变换

//相当于得到的这个值，就是我们的上下方向，通过他来控制对象上下移动或旋转
Input.GetAxis("Vertical");

鼠标横向移动时，-1到1左右

Input.GetAxis("Mouse X");

鼠标纵向移动时，-1到1左右

Input.GetAxis("Mouse Y");

GetAxisRaw方法和GetAxis方法使用相同
但GetAxisRaw方法的返回值只会是-1,0,1，不会有中间值
而GetAxis方法的返回值是有渐变的

练习题

题1、2

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson11_TankMove : MonoBehaviour
{
    public float moveSpeed = 5;
    public float rotateSpeed = 10;

    public Transform head;
    public float headSpeed = 50;
    // Start is called before the first frame update
    void Start()
    {
        
    }

    // Update is called once per frame
    void Update()
    {
        //公式： 前进方向 * 速度 * 时间 * 当前是否移动(-1 ~ 1)
        //ws控制
        transform.Translate(Vector3.forward * moveSpeed * Time.deltaTime * Input.GetAxis("Vertical"));
        //ad控制
        transform.Rotate(Vector3.up * rotateSpeed * Time.deltaTime * Input.GetAxis("Horizontal"));

        //头部转动
        head.Rotate(Vector3.up * headSpeed * Time.deltaTime * Input.GetAxis("Mouse X"));
    }
}
