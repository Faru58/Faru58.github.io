PlayerPrefs概述与基本方法

PlayerPrefs概述与基本方法

PlayerPrefs概述

数据持久化：
将内存中的数据模型转换为存储模型，以及将存储模型转换为内存中的数据模型的统称
（将游戏数据存档到硬盘，硬盘中数据读取到游戏中）

PlayerPrefs：Unity提供的可以用于存储读取玩家数据的公共类

PlayerPrefs基本方法

存储相关

PlayerPrefs的数据存储，类似于键值对存储，一个键对应一个值
unity中提供了3种数据的方法：int float string
键：string类型
值：int float string 对应三种API

直接调用Set相关方法，只会把数据存到内存里
当游戏结束时，unity会自动把数据存到硬盘中
如果游戏不是正常结束的(崩溃等)，数据是不会存到硬盘中的

void Start()
{
    PlayerPrefs.SetInt("myAge", 17);
    PlayerPrefs.SetFloat("myHeight", 155.5f);
    PlayerPrefs.SetString("myName", "堀江");
}

调用Save方法，就会马上存储到硬盘中

void Start()
{
    PlayerPrefs.Save();
}

PlayerPrefs是有局限性的，它只能存3种类型的数据
如果要存储别的类型的数据，只能降低精度或者上升精度来进行存储

void Start()
{
    bool sex = true; 
    PlayerPrefs.SetInt("sex", sex ? 1 : 0);
}

如果不同类型用同一键名进行存储，会进行覆盖

void Start()
{
    PlayerPrefs.SetInt("myAge", 17);
    PlayerPrefs.SetFloat("myAge", 20.2f);
}

读取相关

注意：运行时，只要你Set了对应键值对
即使你没有马上存储Save到本地，也能读出数据

第二个参数：默认值(在得到没有的数据的时候，用它来进行基础数据的初始化)

void Start()
{
    int age = PlayerPrefs.GetInt("myAge",100); 
    float age = PlayerPrefs.GetFloat("myHeight",100); 
    string name = PlayerPrefs.GetString("myName"); 
}

判断数据是否存在

void Start()
{
    if( PlayerPrefs.HasKey("myName") )
    {
        //逻辑
    }
}

删除相关

删除数据

void Start()
{
    //删除指定键值对
    PlayerPrefs.DeleteKey("myName");
    //删除所有存储的信息
    PlayerPrefs.DeleteAll();
}

题2

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Item
{
    public int id;
    public int num;
}

public class Player
{
    public string name;
    public int age;
    public int atk;
    public int def;
    public List<Item> items;


    public void SaveData()
    {
        PlayerPrefs.SetString("玩家", name);
        PlayerPrefs.SetInt("年龄", age);
        PlayerPrefs.SetInt("攻击力", atk);
        PlayerPrefs.SetInt("防御力", def);
        PlayerPrefs.SetInt("道具数", items.Count);

        for (int i = 0; i < items.Count; i++)
        {
            PlayerPrefs.SetInt("道具ID" + i, items[i].id);
            PlayerPrefs.SetInt("道具" + i + "数量", items[i].num);
        }

        PlayerPrefs.Save();
    }

    public void LoadData()
    {
        name = PlayerPrefs.GetString("玩家", "未命名");
        age = PlayerPrefs.GetInt("年龄", 17);
        atk = PlayerPrefs.GetInt("攻击力", 0);
        def = PlayerPrefs.GetInt("防御力", 0);

        int num = PlayerPrefs.GetInt("道具数", 0);
        items = new List<Item>();
        for (int i = 0; i < num; i++)
        {
            Item item = new Item();
            item.id = PlayerPrefs.GetInt("道具ID" + i);
            item.num = PlayerPrefs.GetInt("道具" + i + "数量");
            items.Add(item);
        }
    }
}



public class Lesson1 : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        Player p = new Player();
        p.LoadData();
        print(p.items.Count);
        for (int i = 0; i < p.items.Count; i++)
        {
            print("道具ID" + i + ":" + p.items[i].id);
            print("道具" + i + ":" + "数量" + p.items[i].num);
        }
        
        Item item = new Item();
        item.id = 1;
        item.num = 1;
        p.items.Add(item);
        item = new Item();
        item.id = 3;
        item.num = 3;
        p.items.Add(item);
        p.SaveData();
    }

    // Update is called once per frame
    void Update()
    {
        
    }
}
