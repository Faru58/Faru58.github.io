Transform相关知识点

Transform相关知识点

Transform

游戏对象(GameObject)位移、旋转、缩放、父子关系、坐标转换等相关操作都由它处理
它是unity提供的极其重要的类

Vector3基础

Vector3是用来表示三维坐标系中的一个点或者一个向量

//声明
Vector3 v = new Vector3();
v.x = 10;
v.y = 10;
v.z = 10;

//只传x, y 默认z = 0
Vector3 v1 = new Vector3(10, 10);
//一步到位
Vector3 v2 = new Vector3(10, 10, 10);
//+ - * / 同向量（向量之间相乘除没说）

常用的点

Vector3.zero (0 0 0)
Vector3.left (-1 0 0)
Vector3.right (1 0 0)
Vector3.forward (0 0 1)
Vector3.back (1 0 0)
Vector3.up (0 1 0)
Vector3.down (0 -1 0)

计算两个点之间的距离

Vector3.Distance(v1, v2);

位置

相对世界坐标系原点位置，即通过position得到的位置

this.GameObject.transform

相对于父对象的位置，面板上显示的transform是相对于父对象的

this.transform.localPosition

对象当前的各朝向(这还是得到坐标)

this.transform.forward //对象当前的面朝向
this.transform.up //对象当前的头顶朝向
this.transform.right //对象当前的右手边

注意：位置的赋值不能直接改变x、y、z，即不能单独改变x、y、z某一个值，只能整体改变

若只想改一个值:

  1. 直接赋值

    this.transform.position = new Vector3(20, this.transform.position.y, this.transform.position.z);

  2. 先取出来再赋值

    Vector3 vPos = this.transform.localPosition;
vPos.x = 20;
this.transform.position = vPos;

位移

路程 = 方向 * 速度 * 时间

参数1：表示位移多少
参数2：表示相对坐标系，默认是相对于自己坐标系的

//相对自己坐标系（只能朝着自己的z方向移动）
this.transform.Translate(Vector3.forward * 1 * Time.deltaTime);
//相对世界坐标系（只能朝着世界的z方向移动）
this.transform.Translate(Vector3.forward * 1 * Time.deltaTime, Space.World);

练习题

题1

层数规律: 每一层方块的数量 = 层数 * 层数
坐标规律: 把每一层的左上角认为是开始位置

  1. x: -0.5 * （层数 - 1）
  2. y: -1 * （层数 - 1）
  3. z: 0.5 * （层数 - 1）
    每一层立方体位置规律: 起始位置 + new Vector(编号 % 层数 * 1, 0, 编号 / 层数 * (- 1))
using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class TowerCreate : MonoBehaviour
{
    public int n = 1;

    // Start is called before the first frame update
    void Start()
    {
        //决定层数(第i+1层)
        for (int i = 0; i < n; i++)
        {
            //每一层创建时，确定一个初始的坐标
            Vector3 pos = new Vector3(-0.5f * i, -1 * i, 0.5f * i);
            //实例化立方体，设置位置
            for (int j = 0; j < (i + 1) * (i + 1); j++)
            {
                //实例化立方体
                GameObject objCube = GameObject.CreatePrimitive(PrimitiveType.Cube);
                //设置位置
                objCube.transform.position = pos + new Vector3(j % (i + 1) * 1, 0, j / (i + 1) * (-1));
            }
        }
    }

    // Update is called once per frame
    void Update()
    {
        
    }
}
