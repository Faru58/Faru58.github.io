光源系统基础

光源系统基础

光源系统基础

Type(光源类型)

  1. Spot 聚光灯
    Range：发光范围距离
    Spot Angle：光锥角度
  2. Directional 方向光(环境光)
  3. Point 点光源
  4. Area 面光源
    只在烘焙状态下有用，实时状态下不起用

Color(颜色)

Mode(光源模式)

  1. Realtime 实时光源
    每帧实时计算，效果好，性能消耗大

  2. Baked 烘焙光源
    事先计算好，无法动态变化

  3. Mixed 混合光源
    事先计算+实时计算

Intensity(光源亮度)

Shadow Type

  1. NoShadows 关闭阴影
  2. HardShadows 生硬阴影
  3. SoftShadows 柔和阴影

Cookie(投影遮罩)

Cookie Size(投影遮罩范围)

Draw Halo(球形光环开关)

Flare(耀斑)

Culling Mask(提出遮罩层)

决定哪些层的对象受到该光源的影响

Indirect Multiplier

改变间接光的强度
低于1：每次反弹会使光更暗
高于1：每次反弹会使光更亮

RealtimeShadows

  1. Srength：阴影暗度0~1之间，越大越黑
  2. Resolution：阴影贴图渲染分辨率，越高越逼真，消耗越高
  3. Bias：阴影推离光源的距离
  4. Normal Bias：阴影投射面沿法线收缩距离
  5. Near Panel：渲染阴影的近裁剪面

Render Mode(渲染优先级)

  1. Auto：运行时确定
  2. Important：以像素质量为单位进行渲染，效果逼真，消耗大
  3. Not Important：以快速模式进行渲染

练习题

题1

挂载点光源

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson15_LaZhu : MonoBehaviour
{
    public Light fire;
    public float moveSpeed = 0.2f;
    public float flashSpeed = 0.5f; 
    // Start is called before the first frame update
    void Start()
    {
        
    }

    // Update is called once per frame
    void Update()
    {
        fire.transform.Translate(Vector3.right * moveSpeed * Time.deltaTime);
        if(fire.transform.position.x >= 1 || fire.transform.position.x <= 0.5f)
        {
            moveSpeed = -moveSpeed;
        }
        
        fire.intensity += flashSpeed * Time.deltaTime;
        if(fire.intensity >= 1)
        {
            flashSpeed = -flashSpeed;
        }
        else if (fire.intensity <= 0.5)
        {
            flashSpeed = -flashSpeed;
        }
    }
}

using System.Collections;

using UnityEngine;

public class Lesson15_ChangeDay : MonoBehaviour
{
    public Transform sun;
    public float speed;
    // Start is called before the first frame update
    void Start()
    {
        
    }

    // Update is called once per frame

    }
}
