Camera可编辑参数

Camera可编辑参数

Camera可编辑参数

Clear Flags

如何清除背景

  1. skybox 天空盒(3D游戏)
  2. Solid Color 颜色填充(2D游戏)
  3. Depth only 只画该层，背景透明(多个摄像机叠加配合)
  4. Don’t Clear 不移除，覆盖渲染

Culling Mask

选择性渲染部分层级：可以指定只渲染对应层级的对象

Projection

Perspective 透视模式

  1. FOV Axis： 视场角、轴，决定了光学仪器的视野范围
  2. Field of view： 视口大小
  3. Physical Camera：物理摄像机，勾选后可以模拟真实世界中的摄像机
    焦距： Focal Length
    传感器尺寸： Sensor Type
    传感器尺寸： Sensor Size
    透镜移位： Lens Shift
    闸门配合： Gate Fit

Orthographic 正交摄像机（一般用于2D游戏制作）

Size： 摄制范围

Clipping Planes

裁剪平面距离

Viewport Rect

视口范围： 屏幕上将绘制该摄像机视图的位置
主要用于双摄像机游戏，0`1相当于宽高百分比

Depth

渲染顺序上的深度

Redering path

渲染路径

Target Texture

渲染纹理： 可以把摄像机渲染到一张图上，主要用于制作小地图
在Project右键创建Render Texture

Occlusion Culling

是否启用剔除遮挡

Allow HDR

是否允许高动态范围渲染

Allow MSAA

是否允许抗锯齿

Allow Dynamic Resolution

是否允许动态分辨率呈现

Target Display

用于哪个显示器，用来开发有多个屏幕的平台游戏
