父子关系知识点

父子关系知识点

父子关系

获取和设置父对象

获取父对象

this.transform.parent

直接设置父对象

this.transform.parent = null; //断绝父子关系
this.transform.parent = newParent; //认新的爸爸

API设置父对象 SetParent
参数1：想要设定的父对象
参数2：是否保留世界坐标系的位置、角度、缩放信息
（true：会保留世界坐标系下的状态，和父对象进行计算得到本地坐标系的信息；false：不会保留会直接把世界坐标系的状态赋值到本地坐标系下）

this.transform.SetParent(null); //断绝父子关系
this.transform.SetParent(newParent); //认新的爸爸

抛妻弃子

和自己的所有儿子断绝关系

this.transform.DetachChildren();

获取子对象

按名字查找儿子，找到儿子的transform信息
Find方法能找到失活对象,GameObject.Find是找不到失活对象的

this.transform.Find("儿子名字").name;

注意：只能找到儿子，不能找到孙子
虽然效率要高于GameObject.Find，但前提是必须要知道父亲是谁

遍历儿子

  1. 失活的儿子也会算数量
  2. 找不到孙子，所以孙子不会算数量
    this.transform.childCount;
//通过索引号能得到对应的儿子，如果索引号超过儿子数量的范围，那么直接报错
for(int i = 0; i < transform.childCount; i++)
{
    //逻辑
}

儿子的操作

一个对象判断自己的父亲是不是另一个对象

儿子对象.IsChildOf(父亲对象);

得到自己作为儿子的编号

儿子对象.GetSiblingIndex();

把自己设置为第一个儿子

儿子对象.SetAsFirstSibling();

把自己设置为最后一个儿子

儿子对象.SetAsLastSibling();

把自己设置为指定第几个儿子
注意：如果填的索引值超出范围（负数或大于最大），不会报错，会直接设置成最后一个编号

儿子对象.SetSiblingIndex(索引值);

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public static class Tools
{
    public static void Sort(this Transform obj)
    {
        //把子对象存入list中，然后对list进行排序
        List<Transform> list = new List<Transform> ();
        for (int i = 0; i < obj.childCount; i++) 
        {
            list.Add (obj.GetChild (i));
        }
        //利用list的排序对名字长短进行排序
        list.Sort ((a, b) =>
        {
            if(a.name.Length < b.name.Length)
                return -1;
            else
                return 1;
        });
        //根据list中的排序结果，重新设置每一个对象的索引编号
        for (int i = 0; i < list.Count; i++)
        {
            list[i].SetSiblingIndex(i);
        }
    }

    public static Transform Search(this Transform father, string childName)
    {
        Transform target = null;
        target = father.Find(childName);
        if (target != null) return target;
        for(int i = 0; i < father.childCount; i++)
        {
            target = father.GetChild(i).Search(childName);
            if (target != null) return target;
        }
        return target;
    }
}
