坐标转换知识点

坐标转换知识点

坐标转换

世界坐标转本地坐标

世界坐标系转本地坐标系可以帮助我们大概判断一个相对位置

世界坐标系的点转换为相对本地坐标系的点（受到缩放影响）

this.transform.InverseTransformPoint(Vector3.forward)

世界坐标系的方向转换为相对本地坐标系的方向

  1. 不受缩放影响

    this.transform.InverseTransformDirection(Vector3.forward)

  2. 受到缩放影响

    this.transform.InverseTransformVector(Vector3.forward)

本地坐标转世界坐标

本地坐标系的点转换为相对世界坐标系的点（受到缩放影响）

this.transform.TransformPoint(Vector3.forward)

本地坐标系的方向转换为相对世界坐标系的方向

  1. 不受缩放影响

    this.transform.TransformDirection(Vector3.forward)

  2. 受到缩放影响

    this.transform.TransformVector(Vector3.forward)

        sphere3.transform.position = new Vector3(0, 0, 3) + obj.transform.position;
    }
}
