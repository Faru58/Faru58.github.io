快速排序

快速排序

快速排序

基本原理

选取基准，产生左右标识，左右比基准，满足则换位
排完一次基准定位，左右递归直到有序

基本实现

套路写法：基准值变量，左右游标记录
三层while循环

  1. 游标不停左右移动，重合则结束，结束定基准
  2. 递归排左右，错位则结束
  3. 左右互访，while循环外定基准
using System;

namespace Lesson26_快速排序
{
    internal class Program
    {
        static void Main(string[] args)
        {
            int[] array = new int[] { 8, 7, 1, 5, 4, 2, 6, 3, 9 };
            QuickSort(array, 0, array.Length - 1);
            for (int i = 0; i < array.Length; i++)
            {
                Console.WriteLine(array[i]);
            }
        }

        static void QuickSort(int[] array, int left, int right)
        {
            if (left >= right)
                return;
            int temp = array[left];
            int tempLeft = left;
            int tempRight = right;         

            while(tempLeft != tempRight)
            {
                //先从右边开始，比基准值大，则继续寻找，否则将该右移到左边
                while(tempLeft < tempRight &&
                    array[tempRight] > temp)
                {
                    tempRight--;
                }
                array[tempLeft] = array[tempRight];
                //再从左边开始，比基准值小，则继续寻找，否则将该左移到刚才右边移动的位置
                while (tempLeft < tempRight &&
                    array[tempLeft] < temp)
                {
                    tempLeft++;
                }
                array[tempRight] = array[tempLeft];
            }

            //tempLeft和tempRight相等，插入基准值
            array[tempLeft] = temp;

            //递归左边
            QuickSort(array, left, tempRight - 1);
            //递归右边
            QuickSort(array, tempLeft + 1, right);
        }
    }
}

注意：while循环中的判断条件中，索引值的判断(sortIndex >= 0)一定要在前
若是在后，sortIndex变为-1时(noSortNum < array[sortIndex])中array的索引会直接报错
今后的有关数组的循环若是有改变索引值，因注意让索引值的判断在前

总结

归并排序和快速排序都会用到递归
相同点：
它们都会把数组分成几部分
不同点：

  1. 归并排序递归过程中会产生新数组用于合并；快速排序不会产生新数组
  2. 归并排序是拆分数组完毕后再进行排序；快速排序是边排序边拆分
