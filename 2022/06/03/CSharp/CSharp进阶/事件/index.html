事件

事件

概念

事件是基于委托的存在
事件是委托的安全包裹 让委托的使用更具有安全性
事件是一种特殊的变量类型

声明

访问修饰符 event 委托类型 事件名;

事件的使用：
1.事件是作为成员变量存在于类中
2.委托怎么用 事件就这么勇

事件相对于委托的区别：
1.不能在类外部赋值 但可以通过加（+=）、减(-=)去添加、删除记录的函数
2.不能在类外部调用 只能在类的内部封装调用

注意：
它只能作为成员存在于类和接口以及结构体中

使用事件的原因

1.防止外部随意调用委托
2.防止外部随意置空委托
3.事件相当于对委托进行了一次封装 让其更加安全

练习题

using System;

namespace Lesson13_事件
{
    class Calorifier
    {
        public event Action<int> alarmEvent;
        public int temperature = 0;

        public void Heating()
        {
            int index = 0;
            while (true)
            {
                if (index++ % 99999999 == 0)
                {
                    temperature++;
                    if (temperature >= 100)
                    {
                        break;
                    }
                    else
                    {
                        if (alarmEvent != null)
                        {
                            alarmEvent(temperature);
                        }
                        //alarmEvent = null;
                    }
                    index = 0;
                }
            }           
        }
    }

    class Warner : Calorifier
    {
        public void Warning(int temperature)
        {
            Console.WriteLine("水温已经到{0}°了", temperature);
        }
    }

    class Moniter : Calorifier
    {
        public void Hint(int temperature)
        {
            if(temperature < 95)
            {
                Console.WriteLine("水还未烧开");
            }
            else if (temperature <= 100 && temperature >= 95)
            {
                Console.WriteLine("水已经烧开");
            }
        }
    }

    internal class Program
    {
        static void Main(string[] args)
        {
            Calorifier calorifier = new Calorifier();
            Warner warner = new Warner();
            Moniter moniter = new Moniter();

            calorifier.alarmEvent += warner.Warning;
            calorifier.alarmEvent += moniter.Hint;
            calorifier.Heating();

        }
    }
}
