反射和特性—特性

特性

特性

特性是一种允许我们向程序的程序集添加元数据的语言结构
它是用于保存程序结构信息的某种特殊类型的类

特性提供功能强大的方法以将声明信息与C#代码（类型、方法、属性等）相关联
特性与程序实体相关联后，即可在运行时使用反射查询特性信息

特性的目的是告诉编译器把程序结构的某组元数据嵌入程序集中
它可以防止在几乎所有的声明中（类、变量、函数等等声明）

自定义特性

继承特性基类 Attribute

class MyCustomAttribute : Attribute
{
    //特性中的成员 一般根据需求来写
    public string info;

    public MyCustomAttribute(string info)
    {
        this.info = info;
    }
}

特性的使用

本质上就是在调用特性类的构造函数
写在类、函数、变量的上一行，表示他们具有该特性信息

[MyCustom("构造函数需要传入的成员")]
class MyClass : Attribute
{
    [MyCustom("  ")]
    public int value;

    [MyCustom("  ")]
    public void TestFun([MyCustom("  ")] int a)
    {

    }
}

class Program
{
    static void Main( string[] args )
    {
        MyClass mc = new MyClass();
        Type t = mc.GetType();
        //判断是否使用了某个特性：参数1：特性的类型；参数2：代表是否搜索继承链（属性和时间忽略此参数）
        if(t.IsDefined(typeof(MyCustomAttribute)), false)
        {
            Console.WriteLine("该类型应用了MyCustom特性");
        }
        //取Type元数据中的所有特性
        object[] array = t.GetCustomAttributes(true);

    }
}

限制自定义特性的使用范围

通过为特性类 加特性 限制其使用范围

[AttributeUsage(AttributeTargets.Class | AttributeTargets.Struct, AllowMultiple = true, Inherited = false)]

第一个参数 AttributeTargets - 特性能用在哪些地方
第二个参数 AllowMultiple - 是否允许多个特性实例用在同一个目标上
第三个参数 Inherited - 特性能否被派生类和重写成员继承

系统自带特性-过时特性

过时特性 obsolete
用于提示用户 使用的方法等成员已经过时 建议使用新方法
一般加在函数前的特性

[Obsolete("//", true)]

第一个参数 调用过时方法时提示的内容
第二个参数 true-使用该方法时会报错; false-使用该方法会警告

系统自带特性-调用者信息特性

需要引用命名空间 using System.Runtime.CompilerServices;

CallerFilePath特性 - 被调用的文件路径
CallerLineNumber特性 - 被调用的行数
CallerNumberName特性 - 被调用的函数名

一般作为函数参数的特性

系统自带特性-条件编译特性

条件编译特性 Conditional
它会和预处理指令 #define配合使用

需要引用命名空间 using System.Diagnostics;
主要可以用在一些调试代码上
有时想执行有时不想执行的代码上

[Obsolete("函数名")]

系统自带特性-外部Dll包函数特性

DllImport
需要引用命名空间 using System.Runtime.InteropServices;

[DllImport("***.dll")]

用来标记非.Net(C#)的函数，表明该函数在一个外部的Dll中定义
一般用来调用 C或者C++的Dll包写好的方法

练习题

using System;
using System.Reflection;

namespace Lesson21_反射和特性_特性
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Assembly assembly = Assembly.LoadFrom(@"F:\CSharp学习\CSharp进阶教学\类库测试\bin\Debug\类库测试");
            Type[] types = assembly.GetTypes();
            foreach (Type t in types)
            {
                Console.WriteLine(t);
            }

            Type playerType = assembly.GetType("类库测试.Player");

            object playerObj = Activator.CreateInstance(playerType);

            Console.WriteLine(playerObj);

            FieldInfo[] fields = playerType.GetFields();
            foreach (FieldInfo field in fields)
            {
                Console.WriteLine(field);
            }


            //首先得到自定义特性的type
            Type attribute = assembly.GetType("类库测试.MyCustomAttribute");
            //赋值名字
            FieldInfo fieldStr = playerType.GetField("name");
            if(fieldStr.GetCustomAttribute(attribute) != null)
            {
                Console.WriteLine("非法操作，随意修改name成员");
            }
            else
            {
                fieldStr.SetValue(playerObj, "123123");
            }           
        }
    }
}
