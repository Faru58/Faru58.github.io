文本和按钮控件

GUI控件绘制的共同点

  1. 他们都是GUI公共类中提供的静态函数，直接调用即可
  2. 他们的参数都大同小异
    位置参数：Rect参数（x y位置，w h尺寸）
    显示文本：string参数
    图片信息：Texture参数
    综合信息：GUIContent参数
    自定义样式：GUIStyle参数
  3. 每一种控件都有多种重载，都是各个参数的排列组合
    必备的参数内容是位置和显示信息

文本控件

GUI.Label void型

  1. 基本使用

    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Texture tex;
    public Rect rect;
    public void OnGUI()
    {
        GUI.Label(new Rect(0, 0, 100, 20), "GUI");
        GUI.Label(rect, tex);
    }

}

  2. 综合使用

using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Rect rect1;
    public GUIContent content;
    public void OnGUI()
    {
        GUI.Label(rect1, content);
        //可以获取当前鼠标或者键盘选中的GUI控件对应的ToolTip信息
        Debug.Log(GUI.tooltip);
    }

}
  1. 自定义样式 
    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Texture tex;
    public GUIStyle style;
    public void OnGUI()
    {
        GUI.Label(new Rect(0, 0, 100, 20), "GUI", style);
    }

}

按钮控件

GUI.Button bool型
鼠标在按钮范围内点击下去变为TRUE，其余为FALSE

  1. 基本使用

    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Rect buttonRect;

    public void OnGUI()
    {
        GUI.Button(buttonRect, "GUI");
    }

}

  2. 综合使用

    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Rect buttonRect;
    public GUIContent bottonContent;

    public void OnGUI()
    {
        GUI.Button(buttonRect, bottonContent);
    }

}

  3. 自定义使用

    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class Lesson2 : MonoBehaviour
{
    public Rect buttonRect;
    public GUIContent bottonContent;
    public GUIStyle buttonStyle;

    public void OnGUI()
    {
        GUI.Button(buttonRect, bottonContent, buttonStyle);

        if(GUI.Button(buttonRect, bottonContent, buttonStyle))
        {
            //处理按钮点击的逻辑
            Debug.Log("按钮被点击");
        }
        if(GUI.RepeatButton(buttonRect, bottonContent, buttonStyle))
        {
            //处理长按按钮的逻辑
            Debug.Log("长按按钮");
        }
    }

}
